# 과제2 | 외부 라이브러리 사용 보고서

본 프로젝트는 외부 파이썬 라이브러리인 Pygame을 활용하여, 가상의 2D 환경에서 벽면을 감지하고 스스로 방향을 전환하는 자율주행 로봇의 기초적인 움직임을 구현한 시뮬레이터입니다.


1. 프로젝트 정보 및 라이선스
* 프로젝트 이름: Pygame 기반 2D 자율주행 충돌 회피 시뮬레이터
* 개발 언어: Python 3.8 이상
* 라이선스: MIT License (자유로운 사용 및 수정 가능)


2. 외부 라이브러리 정보 및 선정 이유

# 사용 라이브러리 및 버전
* 라이브러리명: pygame
* 버전: 2.5.2

# 선정 및 버전 선택 이유
* 시각화의 용이성: 로봇공학에서 제어 알고리즘을 테스트할 때 터미널 창의 텍스트 로그보다 화면에 직접 움직임을 띄워 디버깅하는 것이 직관적이라 판단하여 선정했습니다. Pygame은 2D 그래픽 렌더링을 가장 가볍고 빠르게 구현할 수 있는 도구입니다.
* 버전 안정성: pygame 2.x 버전 대는 기존 1.x 버전에 비해 CPU 점유율이 낮고 최신 파이썬 환경과 호환성이 좋습니다. 그중에서도 `2.5.2` 버전은 안정성이 검증되어 과제 수행 중 예기치 못한 크래시나 메모리 누수를 방지하기에 적합하여 선택했습니다.


3. 설치 및 실행 방법

# 의존성 라이브러리 설치
제공된 `requirements.txt` 파일을 이용하여 시뮬레이션 구동에 필요한 라이브러리를 일괄 설치합니다.
```bash
pip install -r requirements.txt
